## [1] 라이브러리 받기

`pip install django-mathfilters` 설치 



## [2] settings.py 추가

mathfilters를 apps에 추가해줘야한다

```python
INSTALLED_APPS = [
    ...
    'mathfilters',
    ...
]
```

## [4] 간단한 연산 코드

```html
<!-- 더하기 -->
{{ 10 | add:1 }}
<!-- 빼기 -->
{{ 10 | sub:1 }}
<!-- 곱하기 -->
{{ 10 | mul:2 }}
<!-- 나누기 -->
{{ 10 | div:2 }}
```
